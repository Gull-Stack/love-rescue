## Technical SEO Checklist

### On-Page
- [ ] Unique title tags (50-60 chars) with primary keyword
- [ ] Meta descriptions (150-160 chars) with CTA
- [ ] H1 with primary keyword (one per page)
- [ ] H2/H3 structure with secondary keywords
- [ ] Internal linking between related pages
- [ ] Image alt text with keywords
- [ ] Schema markup (FAQ, HowTo, Article, Organization)

### Site Structure
- [ ] XML sitemap with all pages
- [ ] robots.txt properly configured
- [ ] Canonical URLs on all pages
- [ ] Breadcrumb navigation
- [ ] Mobile-responsive design
- [ ] Page speed < 3 seconds
- [ ] HTTPS everywhere

### Schema Markup
- Organization schema (homepage)
- FAQ schema (service pages)
- Article schema (blog posts)
- HowTo schema (guides)
- Review/Rating schema (testimonials)
- BreadcrumbList schema (all pages)
